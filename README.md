Proyecto: GestorNotas
Nombre: Diego Alejandro Rielo Cabrera

Descripción breve:
  Aplicación de consola en Java para gestionar notas de texto con diferentes niveles de importancia, 
desarrollada siguiendo un flujo de trabajo profesional con Git y GitHub.

Listado de ramas creadas:
1. master (o main): Rama principal de producción.
2. feature-estructura: Creación de la base del proyecto.
3. feature-alta-notas: Implementación de la creación de notas.
4. feature-validacion-alta: Mejoras de validación de entrada.
5. feature-listado: Funcionalidad para mostrar todas las notas.
6. feature-filtro-importancia: Opción para filtrar notas importantes.

Explicación del conflicto resuelto:
  Se provocó un conflicto intencionado al intentar fusionar feature-listado en master. Ambas ramas 
habían modificado la línea del encabezado del menú principal con textos diferentes.

Resolución: 
  Se realizó una fusión manual utilizando la herramienta de IntelliJ, dejando como 
título final: === GESTOR DE NOTAS - VERSION FINAL ===.

Funcionalidad final 
La aplicación permite:
1. Crear notas con título y contenido.
2. Validar que no haya campos vacíos.
3. Listado completo de notas.
4. Filtrado de notas marcadas como importantes.
